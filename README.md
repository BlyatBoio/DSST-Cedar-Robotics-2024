# DSST-Cedar-Robotics-2024

DSST Cedar is participating in the BEST Robotics program in the 2024 school year, this is our code.

In order to run, you need :
1. VEX Cortex (https://www.vexrobotics.com/276-2194.html)
2. VEX Controler "Joysicks" (https://www.vexrobotics.com/joysticks.html)
3. 2 VEXNet Keys (https://www.vexrobotics.com/276-3245.html)
4. VEX Programing Kit (https://www.vexrobotics.com/276-2186.html)
5. Robot C programing interface (https://www.robotc.net/)
Simply copy/paste code into the Robot C programing application, connect your cortex to your computer, and hit "Download To Robot"
